## 1. Gen 2 Workbook Upgrade

Every Excel Cloud workbook has been upgraded to Gen 2 standard. For clients, this means a cleaner, more consistent experience across all files they receive.

- A new Instructions sheet in every workbook — clear guidance on how to use the file, what the formulas do, and how to get the most out of it with Claude's skills
- Improved structure and navigation, making it easier to find what you need
- Consistent formatting and layout across all workbooks, so moving between files feels familiar
- More comprehensive notes on complex formulas so clients don't need to guess how things work

## 2. Skills Library — Supercharge Workflows with Claude AI

We have built a curated library of 15 Claude AI skills, purpose-built for EXL Cloud workbooks. Skills are small instruction files that tell Claude exactly how to help inside a specific workbook or task — no setup, no technical knowledge required.

### How it works

1. Download a skill file from this GitHub library (one click, free)
2. Load it into Claude following the setup steps provided
3. Claude is instantly trained to help with that specific workbook or task

### The 15 Skills

Skills are organised into four tiers based on plan level.

#### Tier 1 — Foundation
Core standards that define quality benchmarks and best practice conventions for all EXL Cloud workbooks.

| # | Skill | Description |
|---|-------|-------------|
| 1 | EXL Cloud Skills Foundations | Quality benchmarking for skill development and model auditing |
| 2 | EXL Cloud Best Practice Financial Modelling | Formatting, colour coding, and formula standards for Excel models |

#### Tier 2 — Data Infrastructure
Foundation skills that prepare data before any reporting or modelling work begins.

| # | Skill | Description |
|---|-------|-------------|
| 3 | Clean & Validate Data | Automated data cleanup before any reporting or modelling work begins |
| 4 | Trial Balance Mapper | Maps chart of accounts from any accounting system into standardised categories |
| 5 | Period Table Builder | Builds the Period date structure that drives all time-series workbooks |

#### Tier 3 — Narrative Layer
Text-output skills that generate commentary, narratives, and board-ready documents from workbook data.

| # | Skill | Description |
|---|-------|-------------|
| 6 | Monthly Commentary Generator | Generates professional management commentary from P&L and cash flow data |
| 7 | Board Paper Drafter | Assembles a complete board-ready report from commentary and financial data |
| 8 | Cash Flow Narrative | Explains cash movements in plain English across operating, investing, and financing activities |

#### Tier 4 — Workbook Augmentation
Skills that add new sheets and structure to existing workbooks.

| # | Skill | Description |
|---|-------|-------------|
| 9 | Assumption Register Builder | Scans a workbook and extracts all hardcoded assumptions into a structured register |
| 10 | Anomaly Detection | Adds automated PASS/FAIL validation checks to any workbook |
| 11 | Model Documentation Generator | Auto-generates an Instructions sheet covering model structure and usage |
| 12 | Variance Analysis Reporter | Builds a structured sheet comparing actuals to budget, forecast, or prior period |
| 13 | KPI Dashboard Formatter | Builds a one-page executive dashboard with tiles and traffic light indicators |
| 14 | Change Log and Audit Trail | Creates structured change tracking sheets (Change Log and Claude Log) |
| 15 | Meeting Prep Pack | Generates a meeting preparation pack from workbook data |

### Which workbooks have skills

Relevant skills are listed directly inside each workbook's instructions sheet, with a download link. Clients only need to download the skills relevant to the workbooks they use. Skills available can be seen in the feature sheet.

Data pool and data push workbooks do not require skills.

### Skill packages

Skills can be downloaded individually or as a combined package by tier, so you don't need to install more than you need.

| Package | Includes |
|---------|----------|
| Tier 1 combo | Skills 1 and 2 |
| Tier 2 combo | Skills 3, 4 and 5 |
| Tier 3 combo | Skills 6, 7 and 8 |
| Tier 4 combo | Skills 9 to 15 |
